# Todo App - Frontend (Client)

React application for the Todo API frontend.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm start
   ```

## Configuration

- API base URL is configured in `src/service.js`
- Default server URL: `http://localhost:5090`

## Build for Production

```bash
npm run build
```

## Deploy to Static Site (Netlify/Vercel)

1. Run `npm run build`
2. Deploy the `build` folder to your hosting service
